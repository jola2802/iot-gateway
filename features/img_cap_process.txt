1. Verbindung mit dem OPC UA-Server herstellen: 
Über eine definierte URL wird eine Verbindung zu einem OPC UA-Server aufgebaut, der Maschinenbilder bereitstellt.
2. Bildaufnahme starten: 
Durch den Aufruf einer speziellen Methode auf dem OPC UA-Server wird ein Bild von einer Kamera erfasst.
3. Auf Bildaufnahme warten: 
Das Programm überwacht einen Knoten, der den Status der Bildaufnahme anzeigt, und wartet, bis das Bild erfolgreich erfasst wurde.
4. Bild herunterladen Node auf true gesetzt: 
Das erfasste Bild wird vom Server heruntergeladen und auf der Festplatte gespeichert und READ-Node wird auf true gesetzt.
5. Bild hochladen: 
Das heruntergeladene Bild wird an einen Oracle-Server hochgeladen, zusammen mit Zeitstempeln und Identifikationsinformationen.
6. Wiederholen: 
Der Prozess wiederholt sich in regelmäßigen Abständen, um kontinuierlich Bilder zu erfassen und hochzuladen.
